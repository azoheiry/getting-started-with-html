HTML: Hyper Text Markup Language.


- Text content What you see
- Markup what it looks like
- References to other documents

HTML is made of Tags
A tag is made of opening tag and a closing tag
An opening and closing tag with the enclsed content is called an element.
Ex:
bold tag
italic
HTML tags can be nested.
bold and italic tag

HTML Tag Attibutes
Tag can have an attribute

Ex: a web link
google
- Img is another tag that can have an attribute.

- img tag is a Void tag has no closing tag.
- br and p are tags that add a new line

Inline and block tags

br is inline while p is block, block adds an invisble box around the text.

- Span is an inline container. Can be used to hold css or JS code.
- Div is block container that adds a box around the enclosed content.
